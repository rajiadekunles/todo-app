# To-Do List Web App

A simple and clean to-do list app built with vanilla JavaScript, HTML, and CSS.

## ✨ Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- Persistent data using `localStorage`

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (ES6)
